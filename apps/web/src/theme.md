# Theme Tokens (WZ Dropshipping)

Global design tokens are defined in `tokens.css`.

## Brand
- `--brand-500`: primary brand
- `--brand-400`: primary action
- `--brand-300`: hover/accent
- `--brand-200`: secondary accent
- `--brand-100`: light accent

## Usage rules
1. New pages/modules should not hardcode brand hex values.
2. Prefer tokens from `tokens.css`.
3. If a new shared color is needed, add it in `tokens.css` first.
4. Keep visual consistency across Auth / Dashboard / Orders / Sourcing modules.
